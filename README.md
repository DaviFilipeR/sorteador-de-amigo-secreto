O projeto do sorteados de amigo secreto é um projeto usado no curso de prática da lógica de programação na Alura. Ele é usado para motivar o aluno a praticar a lógca de programação e praticar conceitos principalmente relacionados a Arrays.

<img src=https://github.com/user-attachments/assets/5d7e246e-b23c-4d50-a870-94d2ceb3a6bb alt="Gif de usuário digitando os nomes para serem sorteados. Os nomes são Álvaro, João Vitor, Thiago, Davi e Luiz, respectivamente. O projeto sorteia os nomes fazendo com que Thiago tire Álvaro, Álvaro tire Luiz, Luiz tire Davi, Davi tire João Vitor e João Vitor tire Thiago. Depois o usuário clica no botão reiniciar e o jogo volta ao estado inicial.">
